# 飞机大战 - 部署文档

## 运行环境

- 现代浏览器（Chrome 80+, Firefox 75+, Safari 14+, Edge 80+）
- 无需服务器，直接打开 HTML 文件即可运行

## 运行方式

1. 下载 `index.html` 文件
2. 用浏览器直接打开即可开始游戏

或者使用本地服务器：

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

## 游戏操作

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动飞机 |
| 空格 / J | 射击 |
| P | 暂停/继续 |

手机端自动显示触屏虚拟按钮。

## 游戏机制

- 5 个关卡，难度递增
- 三种普通敌机：小型（快速低血）、中型（会射击）、大型（高血量）
- Boss 敌机（第2关起出现）
- 四种道具：加速(S)、护盾(D)、速射(R)、全屏炸弹(B)
- 子弹可以对撞抵消
- 本地存储最高分
