# 飞机大战 - DeepSeek Version

飞机大战射击游戏，HTML5 Canvas 单文件实现。

## 技术栈

- HTML5 Canvas + 原生 JavaScript
- Web Audio API 音效
- 无外部依赖

## 目录结构

```
plane-battle-deepseek-version/
├── index.html   # 游戏主文件
└── README.md    # 项目说明
```

## 如何运行

直接在浏览器中打开 `index.html` 即可。

## 操作说明

- **WASD / 方向键**：控制飞机移动
- **空格 / J**：射击
- **P**：暂停/继续

## 游戏特性

- 三种敌机：小型、中型、大型
- 四种道具：加速(S)、护盾(D)、速射(R)、全屏炸弹(B)
- 星空滚动背景
- 5 关难度递增
- 本地最高分记录
- 音效反馈
